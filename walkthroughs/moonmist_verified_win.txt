# Moonmist is scoreless. It is a V3 "time" game (header flags1 bit 1): the
# status-line globals hold the CLOCK, and the game's own source names them
# exactly that way (misc.zil: GLOBAL SCORE starts at 19 = 7 p.m. and is the
# HOURS counter; GLOBAL MOVES is the minutes, rolling the hour at 59). There
# is no numeric score global at all, so no _SCORE_VAR_OVERRIDES entry is
# possible - the "21" above is simply the 9 p.m. clock hour at the end.
# Instead the game tracks three accomplishments (V-SCORE in verbs.zil):
# TREASURE-FOUND, EVIDENCE-FOUND leading to an arrest (CONFESSED), and
# FOUND-COSTUME. Only when all three are true does V-SCORE - which the
# FINISH routine calls at the end - print "[Congratulations, Ms. Smith!
# You've won the game!]", and that line is this walkthrough's WIN_TEXT:
# it cannot appear in any lesser ending.
#
