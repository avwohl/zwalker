# Adaptive @-route for the ZORKIE build of zork2 (renovated source).
# Derived from walkthroughs/zork2_verified_400.txt (official Release-63 route,
# seed 2). Baked RNG outcomes (carousel spins, Low Room coin flips, take
# fumbles, princess/wizard daemon timing, treasure-ferry trips) become
# adaptive steps handled by solve_zork2_zorkie_adaptive.py, which records the
# actual command stream for deterministic replay via scripts/replay_solve.py.
#
# Syntax:  cmd                      plain step
#          cmd @@ Room [| tries]    checked step, retried while stuck in place
#          @take noun | DESC        take with ITAKE-fumble retries, VM-verified
#          @spin Room               escape the spinning Carousel Room
#          @lowexit                 escape the post-button Low Room to Tea Room
#          @waituntil Room | cap    wait for a daemon (balloon) to arrive
#          @waithave DESC | cap     wait until the object is carried
#          @ferry                   loop treasures Guarded Room -> Pentagram
#          @give noun | DESC        take+give to the demon, VM-verified
#          @wand                    poll the demon/wizard for the wand
