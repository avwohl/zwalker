# Moon-Shaped (competition2006 / Release 1) -- winning route derived from the R2 guide.
#
# NOTE ON VERIFICATION: This route DETERMINISTICALLY reaches the game's BEST ending
# (ending 1 of 8: the curse is lifted, both father and grandmother are saved) on every
# seed. The game is Inform 7 and ends via "end the story finally saying ...", printing
# the marker:  *** (This is ending 1 of 8) ***
# scripts/try_route.py detects a win with WIN_RX (won|win|victor|congratulat|success|
# you did it|the end|finis|complete). This game's marker contains none of those words,
# so try_route prints "== NO WIN" even though this is a genuine Inform victory ending.
# All 8 endings use the same "(This is ending N of 8)" format, so no route can trigger
# WIN_RX for this game -- it is a harness win-detection gap, not a route defect.
#
# TWO commands use a trailing period ("southwest." / "east.") ON PURPOSE: those rooms
# (Firefly Glen, Marshy Patch) contain the phrase "way is blocked" in their room text,
# which trips a false-rollback in GameWalker.try_command (its object-based room-change
# detection returns a constant for this game, so a movement into such a room is wrongly
# treated as "blocked" and rolled back). Inform strips the trailing '.', so the move
# still happens in-game, while the walker no longer classifies it as a direction and
# skips the bogus rollback.
#
