Download script example:
  wget https://ifarchive.org/if-archive/games/zcode/photopia.z5 -P games/

Run zwalker test:
  python -m zwalker explore games/photopia.z5 --ai --ai-backend anthropic -j games/photopia_walkthrough.json

Status tracking:
- zwalker_status: Did zwalker successfully explore and generate walkthrough?
- zorkie_status: Did zorkie (our compiler) produce a playable .z file?
- z2js_status: Did z2js interpreter play through successfully?
