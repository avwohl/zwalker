## Download Instructions

### Using wget

Download a single game:
```bash
wget https://ifarchive.org/if-archive/games/zcode/photopia.z5 -P games/zcode/
```

### Using the download script

The project includes a download script:
```bash
python games/download_games.py
```

This will download ~79 games to `games/zcode/` covering:
- 27 Infocom classics (z3, z4, z5)
- 52 modern IF games (z5, z8)

### Manual download

All games are freely available from:
- **IF Archive**: https://ifarchive.org/indexes/if-archive/games/zcode/
- **Eblong Infocom**: https://eblong.com/infocom/gamefiles/
- **IFDB**: https://ifdb.org/

## Archive Sources

### Primary Archives

1. **IF Archive** (main repository)
   - URL: https://ifarchive.org/
   - Games: https://ifarchive.org/indexes/if-archive/games/zcode/
   - Most complete collection of modern IF games

2. **Eblong Infocom Archive**
   - URL: https://eblong.com/infocom/gamefiles/
   - All original Infocom games with release numbers
   - Maintained by Andrew Plotkin (Zarf)

3. **IFDB** (Interactive Fiction Database)
   - URL: https://ifdb.org/
   - Metadata and ratings
   - Top 100 list: https://ifdb.org/viewlist?id=k7rrytlz3wihmx2o

### Reference

- Complete game list: `games/game_list.txt` (100+ games with URLs)
- Download script: `games/download_games.py` (79 priority games)
